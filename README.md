🏔️ Pé na Aventura - Agência de Viagens
Projeto de um site completo para uma agência de ecoturismo e expedições, desenvolvido como parte do meu aprendizado prático em Desenvolvimento Web.

🚀 Sobre o Projeto
Este site foi criado para a Pé na Aventura, focando em uma experiência de usuário imersiva, com design moderno e interativo. O projeto demonstra a transição da minha carreira de vendas para a tecnologia, aplicando conceitos de design responsivo e manipulação de estilos.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica das páginas.

CSS3: Estilização personalizada, animações de hover e layouts em Grid/Flexbox.

Tailwind CSS: Utilizado para prototipagem rápida de componentes.

Google Fonts: Fontes Amatic SC (identidade visual) e Inter.

📂 Funcionalidades Atuais
 Página Inicial: Galeria de fotos e proposta de valor.

 Página de Equipe: Apresentação dos fundadores com cards interativos.

 Próximos Destinos: Catálogo de viagens com efeito de overlay e zoom dinâmico ao passar o mouse.

 Navegação Completa: Menu funcional interligando todas as seções.
